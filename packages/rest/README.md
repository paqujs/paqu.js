# @paqujs/rest

REST module for paqu.js.

## Installation

Node.js 16.x or higher is required.

```bash
npm install @paqujs/rest
yarn add @paqujs/rest
pnpm add @paqujs/rest
```

## Usage

```ts
import { REST } from '@paqujs/rest';

const rest = new REST();

rest.<method>('https://example.com').then(console.log);
```
